# Siddhaarth Chandran — Portfolio

Live at **[siddhaarth.vercel.app](https://siddhaarth.vercel.app)**

---

## About

Personal portfolio built from scratch — no templates, no frameworks, no dependencies. Pure HTML, CSS, and JavaScript.

Desktop version is a full-screen cinematic experience with painting-based atmospherics, section-by-section navigation, and light/dark mode. Mobile version is a separate, independently designed layout that auto-detects the device and serves accordingly.

---

## Structure

```
portfolio/
├── index.html       # Desktop site
├── mobile.html      # Mobile site (auto-redirected from index.html)
├── style.css        # Desktop styles
├── script.js        # Desktop navigation + interactions
└── images/          # Paintings and demo assets
```

---

## Features

**Desktop**
- Section-by-section scroll navigation with JS-driven transitions
- Interlude pages with classical painting backgrounds
- Light / dark mode with full theme switching
- Custom scroll progress indicator
- Certificate overlay with verification links
- Demo video lightbox per project

**Mobile**
- Fully independent layout — not a responsive patch
- Native scroll, sticky nav, hamburger menu
- Same content, same aesthetic, built for touch
- Auto-detected via user agent + screen width
